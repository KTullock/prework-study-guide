HTML

  • HTML acts as the skeleton or foundation for the website.
  • The head element contains information about the webpage.
  • The head element talks to the browser and is mostly unseen by the user.
  • The body elements represents the visible content shown to the user.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Class selectors, such as "card", can add style to a specific element.

Git

  • GitHUB: cloud-based repository for coding projects
  • Git/Git Bash: working dierctory
  • Git Desktop: a place to locally work on projects without affecting the main branch
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git checkout branch-name: switches to that branch
  • git add -A: saves changes to branch
  • git commit -m "message": describes changes made to the branch
  • git pull origin main: makes sure the local branch is up to date
  • git push origin 'branch name': pushes changes to GitHub

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
